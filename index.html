heading tags

  4. dh4efines the most important heading.

paragraph tags
A paragraph always starts on a new line, and is usually a block of text.
  • p tag always starts on a new line, and browsers automatically add some white space
  • With HTML, you cannot change the display by adding extra spaces or extra lines in your HTML code
  • hr tag defines a thematic break in an HTML page, and is most often displayed as a horizontal rule.
  • br tag element defines a line break.
  • pre tag element defines preformatted text.
formatting tags
these contains several elements for defining text with a special meaning.
  1. b - Bold text
  2. strong - Important text
  3. i - Italic text
  4. del - Deleted text
  5. ins - Inserted text
block and inline elements
these elements has a default display value, depending on what type of element it is.
  • A block-level element always starts on a new line
  • A block-level element always takes up the full width available
  • An inline element does not start on a new line.
  • An inline element only takes up as much width as necessary.
image tag
Images can improve the design and the appearance of a web page.
  1. tag is empty, it contains attributes only, and does not have a closing tag.
  2. src - Specifies the path to the image
  3. alt - Specifies an alternate text for the image
anchor tag
this tag defines a hyperlink, which is used to link from one page to another.
  • An unvisited link is underlined and blue
  • A visited link is underlined and purple
  • An active link is underlined and red
url structure
URL is another word for a web address
  1. URL can be composed of words, or an Internet Protocol (IP) address
  2. Uniform Resource Locator (URL) is used to address a document (or other data) on the web.
  3. If a URL contains characters outside the ASCII set, the URL has to be converted.
list tag
lists allow web developers to group a set of related items in lists.
  • An unordered list starts with the ul tag. Each list item starts with the li tag.
  • An ordered list starts with the ol tag. Each list item starts with the li tag.
  • A description list is a list of terms, with a description of each term.
